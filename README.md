## Sobre mim

Desenvolvo projetos web completos, transformando ideias e necessidades de negócio em interfaces funcionais, responsivas e prontas para produção.

Tenho experiência prática com:

- Desenvolvimento de sites institucionais e landing pages;
- Aplicações com React, Next.js e TypeScript;
- Criação de componentes reutilizáveis e design systems;
- Interfaces responsivas para mobile, tablet e desktop;
- Animações avançadas e experiências baseadas em scroll;
- Integração com APIs, serviços externos e links de conversão;
- Otimização de imagens, SEO técnico e metadados;
- Testes visuais, testes de responsividade e validação de builds;
- Versionamento, commits e publicação de projetos com Git e GitHub.
